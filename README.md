Task Management Dashboard

This project is a Task Management Dashboard built with React.js. It allows users to manage their tasks, including adding, editing, filtering, and marking tasks as completed or pending. The design follows a modern and minimalistic approach with soft pastel tones and responsive layouts.
Features

    Add Tasks: Add tasks with an optional due date.
    Edit Tasks: Update task details seamlessly.
    Delete Tasks: Remove tasks that are no longer needed.
    Filter Tasks: Switch between different views like All, Completed, Pending, and Overdue tasks.
    Responsive Design: A clean, mobile-friendly UI with intuitive controls.

Prerequisites

Make sure you have the following installed:

    Node.js (LTS version recommended)
    npm or yarn



    How to Get Started
    1.Clone the Repository:
    git clone https://github.com/ayush362/TaskManagementDashboard
]
    2.Navigate to the Project Directory:
    cd TaskManagementDashboard

    3.Install Dependencies:
    npm install
    or
    yarn install

    4.Start the Development Server:
    npm start
    or
    yarn start    

    5.Open the App: Once the server is running, navigate to http://localhost:3000 in your browser to view the dashboard
